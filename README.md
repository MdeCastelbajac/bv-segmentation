## AUTOMATED BLOOD VESSEL SEGMENTATION ON CD3-STAINED WSI

A research project.

#### REQUIREMENTS
The built-in functions require `rasterio` and `aicspylibczi` libraries for `tif` and `czi` file management.
The code was executed with `python 3.11`.
